## Repository Purpose

The goal of this repository is to keep a structured record of my Labs which i am doing for this course. 
Each lab contains commands, scripts, terminal outputs, and visual proof of completed tasks.

Main skills practiced include:

- Linux command-line workflow
- Bash scripting and automation
- User permissions and access control
- Task scheduling using cron
- Package installation and management
- Web server configuration
- Security testing fundamentals
- Log collection and analysis
